Pick’em App

Purpose:
Pick’em App helps users analyze lottery number trends and history for Pick 3 and Pick 4 games. Users can see past occurrences of numbers and make more informed selections.

Features:

Number History Lookup – View all previous draws for selected numbers.

Draw Details – For each number, see:

Total occurrences

Draw type (Midday or Evening)

Number of draws

Last draw date

Current gap (since last drawn)

Previous gap

Average gap

Longest gap

Draw Listing – Displays all draw dates, day of the week, and draw type for selected numbers.

Number Generator – Generates number combinations for both Pick 3 and Pick 4 games.

Tech Stack:

Frontend: React Native / Expo

Data Storage: Local device storage (no backend needed)
