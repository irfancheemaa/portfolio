NFL Game Prediction App

Purpose:
This app predicts weekly NFL game winners using a machine learning model on the backend and displays the probability of each team winning.

Example:

Team A: 49% chance of winning

Team B: 51% chance of winning

Features:

Game Predictions – ML-based weekly game winner predictions.

Community Picks – Users can make their own predictions including:

Winning team

Final score

Over/Under

Personal win probability estimate

Community Sentiment – Aggregates user predictions to show a community breakdown.

Social Interaction – Compare picks with friends and other users.

Tech Stack:

Frontend: React Expo

Backend: FastAPI hosted on Render

HTTP Requests: Axios

Machine Learning Model: XGBoost

Deployment: Frontend hosted on GitHub Pages
